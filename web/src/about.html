Sobre o projeto!

O projeto se iniciou devido a uma grande dor encontrei na área de dados das empresas: coletar e tratar dados! Sempre que precisava coletar dados de um site ou precisava conectar com um API de terceiro para baixar dados era aquele trabalho incansável de criar e recriar a roda diversas vezes. Dado isso resolvi criar uma biblioteca para ser um "toolkit" pra dados, e a ideia desse kit de ferramentas é não só extrair dados externos, mas também facilitar a visualização e pós processamento desses dados, seja propondo gráficos de forma inteligente com base nos dados ou seja sugerindo um modelo matemático para encontrar padrões nesses dados.
De qualquer forma, a ideia do projeto foi crescendo e crescendo e hoje o objetivo é não só fazer um toolkit para pessoas técnicas, mas sim evoluir isso e criar uma ferramenta para que pessoas leigas possam ter acesso a mais dados de forma desburocratizada.
